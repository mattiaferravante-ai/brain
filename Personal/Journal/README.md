# 📓 Journal

Journaling libero — nessuna cadenza fissa. Scrivi quando senti il bisogno.

## Convenzione file

`YYYY-MM-DD_titolo-breve.md` — es. `2026-06-02_dopo-il-viaggio-croazia.md`

## Template rapido

```
# [Titolo o data]

**Contesto:** cosa sta succedendo in questo periodo

**Cosa ho in testa:**

**Cosa sta andando bene:**

**Cosa mi pesa:**

**Cosa voglio cambiare / provare:**
```

**Tag:** `#journal` `#riflessioni`
