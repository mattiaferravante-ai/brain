## Fasi tipiche di un progetto Odoo

1. **Kick-off** — raccolta requisiti AS-IS, stakeholder mapping
2. **Analisi** — redazione AF (AS-IS / TO-BE / Gap), approvazione cliente
3. **Setup** — configurazione base Odoo, COA, multi-company se necessario
4. **Sviluppo custom** — moduli, integrazioni, report
5. **UAT** — test book, esecuzione test, bug tracking
6. **Go-live** — migrazione dati, formazione utenti, hypercare
7. **Post go-live** — bug fix, ottimizzazioni, evolutive
