# Archive — Progetti non partiti o conclusi

Contiene i progetti che sono stati archiviati: gare perse, progetti
cancellati prima del kick-off, o progetti conclusi.

Per archiviare un progetto da `projects/`: usare il comando `/archive-project`.

## Progetti archiviati

| Progetto | Cliente | Stato | Motivo | Archiviato |
|----------|---------|-------|--------|------------|
| *(nessuno ancora)* | | | | |

## Ciclo di vita — stati possibili

| Tag | Significato |
|-----|-------------|
| `#bid` | Gara in corso, esito incerto — progetto non ancora avviato |
| `#active` | Progetto avviato, in delivery |
| `#closed` | Concluso con successo (go-live avvenuto) |
| `#archived` | Non partito (gara persa o progetto cancellato prima del kick-off) |
